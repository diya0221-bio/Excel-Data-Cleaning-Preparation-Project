Excel Data Cleaning & Preparation Project

This project focuses on cleaning and preparing a raw, messy business dataset using Microsoft Excel. The objective was to transform unstructured and inconsistent data into a clean, standardized, and analysis-ready format suitable for reporting and dashboards.

•	Data Cleaning Steps Performed

•	Fixed broken and unreadable date values

•	Standardized text fields (client, department, region)

•	Cleaned and formatted currency values

•	Handled missing values using logical placeholders

•	Corrected numerical columns for accurate calculations

•	Structured the dataset for further analysis

Before vs After

•	Before: Raw, inconsistent, unreadable data with formatting and missing value issues

•	After: Clean, structured, standardized, and analysis-ready dataset
